## 2. Technical Decisions Made

| Decision | Reason |
|----------|--------|
| **Next.js 14.2.35 (not 15)** | Stable version; NextAuth v5 beta has better 14.x compatibility |
| **Manual project setup** | Existing docs folder required careful integration |
| **Tailwind 3.4 (not 4.x)** | Better shadcn/ui compatibility; v4 is still in beta |
| **Placeholder Prisma schema** | Full schema in Prompt 2; needed for postinstall |
| **ESLint 8.x** | Next.js 14 uses ESLint 8; v9 has breaking changes |
| **React 18.x** | Stable version; React 19 not needed for this project |
| **Geist font** | Modern, professional font from Vercel |
| **Sonner for toasts** | Better DX than react-hot-toast; shadcn compatible |
| **Custom badge variants** | Added positive/neutral/negative for sentiment display |
